# Assignment 3

## File Name

`a3.html`

## Description

This assignment demonstrates the use of:

* `iframe` with the `srcdoc` attribute
* External website link opening in a new tab
* Internal page link opening in the same tab
* Link to another local HTML file
* Hyperlinks inside an iframe
* Video download link using the `download` attribute

## Features

1. Displays content inside an iframe.
2. Opens Google in a new browser tab.
3. Navigates to another page in the same tab.
4. Opens a local HTML file.
5. Provides a link to download a video file.

## Technologies Used

* HTML5

## Output

The webpage contains an iframe with multiple links and a download button for a video file.
